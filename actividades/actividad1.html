Series de Tiempo

Una serie de tiempo es una secuencia de datos medidos en puntos sucesivos en el tiempo con intervalos uniformes

Formalmente Yt:{y1,y2,,yT} donde conocemos los valores de la serie hasta T

Componentes de una Serie de Tiempo

  1. Tendencia: Direccion general a largo plazo
  2. Estacionalidad: Patrones que se repiten en intervalos regulares (Por ejemplo: diarios, mensuales & anuales)
  3. Ciclos: Fluctuaciones a largo plazo que no son de naturaleza estacional
  4. Ruido: Variabilidad aleatoria e impredecible

Metodos para descomponer series de tiempo

  1. Clasica: Separando la serie en sus componentes aditivos o multiplicativos

  2. STL (Seasonal and Trend decomposition using Loess): Es un metodo robusto que utiliza regresion local para estimar tendencia y estacionalidad

from statsmodels.tsa.seasonal import STL

stl = STL(precios_cierre,period=252)
result = stl.fit()

data = {
    "Original":precios_cierre,
    "Tendencia":result.trend,
    "Estacionalidad":result.seasonal,
    "Residuo":result.resid
}
